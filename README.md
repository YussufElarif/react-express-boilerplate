### Install
Install necessary packages to get your environment set up
```
npm install
```

### Mongo (Optional)
Set up [Mongo](https://docs.mongodb.com/manual/installation/) and run mongod.
```
mongod
```
Uncomment code block in following paths
```
 - app.js <-- Here
    - routes/
        - controllers/
            - product.js  <-- Here
```
### Start
```
npm start
```
Goto http://localhost:3300
